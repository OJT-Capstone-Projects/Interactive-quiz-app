# Quiz Master – Interactive Quiz App

A full-screen interactive quiz application built using pure HTML, CSS, and JavaScript. The application allows users to enter their name, select a subject, and attempt timed multiple-choice quizzes with real-time feedback, score tracking, and a modern responsive UI with theme switching.

---

## Features

- Name input screen for user personalization  
- Subject selection (JavaScript, HTML, CSS)  
- Full-screen quiz interface (no card-based layout)  
- Timed questions with auto-next functionality  
- Instant correct/wrong answer feedback  
- Score tracking system  
- Progress bar indicator  
- Light/Dark theme switcher  
- Fully responsive design (mobile, tablet, desktop)  
- Restart and replay functionality

## How It Works

1. Enter your name  
2. Select a subject  
3. Answer timed multiple-choice questions  
4. Get instant feedback after each question  
5. View final score and percentage result  
6. Restart or choose another subject  

---

## Theme System

The app supports two themes:

- Peach Theme (Default)  
- Dark Mode  

Implemented using CSS variables and a JavaScript toggle switch.

---

## About

Quiz Master is a beginner-friendly project designed to demonstrate state management, DOM manipulation, and interactive UI development using pure JavaScript without any frameworks.
